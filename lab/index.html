The Lab

This is the coding gym. Small exercises with a focus: loops, array methods, refactors, debugging, test-driven thinking.

Starter lab ideas

  • Fix this broken triage algorithm (if/else to clean functions).
  • Refactor a “spaghetti” block into clean, reusable code.
  • Translate "protocol" steps into a state machine.

Each lab will get its own page or repo link with a starter file and a guided solution.
